关于我们

      北京中农四方农业规划设计研究院有限公司专业从事现代农业规划设计、休闲农业规划设计、都市农业规划设计、农业工程施工、园艺专业技术服务，是农业嘉年华、农业主题公园策划设计、投资建设的全产业链运营商。

      我公司与中国农业大学、中国农科院、北京农林科学研究院、北京 农学院、北京林业大学等科研院所有着长期的科研课题、项目开发及成 果转化合作，在实际的项目开展中为客户提供最权威的专业技术支持。

      我们与美国、荷兰、以色列等国际农业技术发达国家相关机构建立了良 好的合作关系，应用最先进的现代农业技术及理念为项目实施服务。

设计总监

岗位职责：

  1. 带领并指导全案项目的创意构思及执行，保证并监督设计部的作品质量；
  2. 配合其他部门对公司品牌树立及产品宣传进行创意构思及创作指导；
  3. 负责规划项目的策略制定，景观作品的创意设计，同时能带领创作团队进行工作，善于利用、引导团队开发创作；
  4. 负责对农业规划设计人员进行专业培训和指导；
  5. 协调规划设计部与其他部门的工作关系；
  6. 定期做出部门总结报告和发展建议，协同公司发展制定部门成本规划。

任职资格：

  1. 风景园林、环境艺术、景观设计等相关专业，正规院校毕业全日制本科以上学历;
  2. 三年以上同等职位相关工作经验，有一定的手绘表现能力及美术功底，具有大型设计院、景观公司相关工作经验者优先；
  3. 具有创新想法、创作视野开阔；
  4. 具有良好的团队合作精神、沟通、协调能力，具备诚实、认真、负责的工作精神；
  5. 具有良好的专业能力，有可以追溯的成功经验；
  6. 创作欲望强烈，创意构思独特、表现力强，具有良好的团队协作精神。

任职资格：

  2. 三年以上景观方案设计相关工作经验，有一定的手绘表现能力及美术功底，具有大型设计院、景观公司相关工作经验者优先；
  3. 具有创新想法、工作中注重细节，熟练运用AutoCAD、PS、SU、PPT等相关软件；
  4. 具有良好的团队合作精神、沟通、协调能力，具备诚实、认真、负责的工作精神；

行政专员

岗位职责：

  1. 协助综合部经理完成公司行政事务性工作及部门内部日常事务工作。

任职资格：

  1. 协助上级制定行政、人事及企业管理工作发展规划和计划;
  2. 协助审核、修订行政管理规章制度，进行日常行政工作的组织与管理;
  3. 协助高级管理人员进行财产、内务、安全管理，为其他部门提供及时有效的行政服务;
  4. 考核和指导行政部工作 协助承办公司相关法律事务;
  5. 参与公司经营事务的管理和执行工作;

高级园艺技术师

岗位职责：

  1. 协助总经理做好分管部门年度发展战略、年度经营计划及人才发展战略；
  2. 负责公司整体园艺技术指导，园艺技术实施；
  3. 负责制定园区育苗计划、技术方案、施工方案；施工现场技术指导及要点把控；
  4. 建立部门技术工作流程和技术标准；组织部门员工，对内外技术交流，技术培训，不断优化技术工艺，及时调整工作流程和技术标准；
  5. 对公司其他部门提供园艺技术支持。

任职资格：

  1. 本科及以上学历；
  2. 园艺、园林、农学等相关专业；
  3. 3年以上相关工作经历；
  4. 心胸宽广，抗压能力强。

园艺技术师

岗位职责：

  1. 现场技术指导：负责生产基地作物栽培田间管理的技术实施工作-果树、花卉技术指导，修剪及植物病虫害防治；
  2. 负责生产技术方案的提报工作，修改和完善种植操作规程，总结种植技术；
  3. 负责对生产工人进行技术指导，协调和组织农业技术培训 ，并检查验收其工作结果；
  4. 负责田间作物长势以及病虫害、草情的监测预报工作；
  5. 负责病虫害、杂草等的防治工作；
  6. 协助领导对用工、生产过程进行管理实施内部检查；
  7. 组织种植实验。

任职资格：

  1. 本科及以上学历；
  2. 园艺、园林、农学等相关专业；
  3. 3年以上农作物种植相关工作经历；
  4. 心胸宽广，抗压能力强。
